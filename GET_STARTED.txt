FEATURES OVERVIEW
═══════════════════════════════════════════════════════════════════════════

DOCUMENT PROCESSING:
  • Upload PDF or text files
  • Automatic text extraction with pypdf
  • Fallback to Claude Vision OCR for scanned documents
  • Structured field extraction (dates, parties, amounts, etc.)

DRAFT GENERATION (5 Types):
  • Case Fact Summary - Concise overview of key facts
  • Internal Memo - Professional internal analysis
  • Notice Summary - Structured notice-related summary
  • Document Checklist - Verification checklist
  • Title Review - Title-specific analysis

SEMANTIC SEARCH:
  • ChromaDB vector embeddings
  • Similarity-based document retrieval
  • Evidence citation in generated drafts

LEARNING SYSTEM:
  • Tracks user edits to generated drafts
  • Extracts reusable style rules from edits
  • Applies learned rules to future drafts
  • Maintains rule frequency statistics

═══════════════════════════════════════════════════════════════════════════
API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════

UI ENDPOINTS:
  GET  /                    - Homepage with file upload
  POST /process             - Process document and generate draft
  POST /edits               - Submit edited draft for learning

API ENDPOINTS:
  GET  /api/health          - Health check
  POST /api/process         - Process document (JSON)
  POST /api/draft           - Generate draft from docs (JSON)
  GET  /api/documents       - List uploaded documents
  GET  /api/drafts          - List generated drafts
  GET  /api/rules           - Get learned style rules
  POST /api/edit            - Submit edit for learning

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Q: "ANTHROPIC_API_KEY is not configured"
A: Set your API key: export ANTHROPIC_API_KEY="your-key"

Q: "Port 8000 already in use"
A: Change port: python -m uvicorn app.main:app --port 8001

Q: Document processing fails
A: Ensure you have a valid Anthropic API key set

Q: PDF extraction doesn't work on specific PDFs
A: The system automatically falls back to Claude Vision OCR
